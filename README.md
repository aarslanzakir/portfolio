## About This Project

A modern, responsive personal portfolio website built with **Next.js 15**, **React 19**, **TypeScript**, and **Tailwind CSS 4**. Features smooth scroll-reveal animations, mobile-first responsive design, and a warm earthy color palette.

### Highlights

- **Fully Responsive** — Optimized for all devices from mobile (320px) to desktop (1920px+)
- **Smooth Animations** — IntersectionObserver-powered scroll reveals with staggered transitions
- **Accessibility** — Respects `prefers-reduced-motion` for users who prefer minimal animation
- **Performance** — Server-side rendering, optimized fonts, and lightweight bundle
- **Modern Stack** — Next.js 15 App Router, React 19, Tailwind CSS 4, TypeScript 5

## Getting Started

### Prerequisites

- **Node.js** 18.17 or later
- **npm**, **yarn**, **pnpm**, or **bun**

### Installation

```bash
# Clone the repository
git clone https://github.com/aarslanzakir/arslan-portfolio.git

# Navigate to the project
cd arslan-portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

### Build for Production

```bash
npm run build
npm start
```
